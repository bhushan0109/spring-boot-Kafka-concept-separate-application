<h1 align="center">
   Kafka Consumer
</h1>

This API is responsible for consuming the means of payment in the queue of Apache Kafka

## Technologies
 
- [Spring Boot](https://spring.io/projects/spring-boot)
- [Spring MVC](https://docs.spring.io/spring-framework/reference/web/webmvc.html)
- [Spring Kafka](https://docs.spring.io/spring-kafka/reference/html/)
- [Spring Data JPA](https://spring.io/projects/spring-data-jpa)
- [Docker](https://docs.docker.com/get-started/)

## Practices adopted

- SOLID
- API REST
- Sending data in Message Broker queues
- Dependency Injection
- Automated tests
- Containerization with docker
- Queries with Spring Data JPA

## Get started

To upload the API in an automated way, use the repository: [kafka-messaging](https://github.com/WeversonL/kafka-messaging)
